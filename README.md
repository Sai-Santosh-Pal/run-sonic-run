# RUN SONIC RUN!

- A game made in Kaplay.js
- To download and contribute
  1. Clone the repo
  2. Run ```npm install```
  3. Then ```npm run dev```
  4. Now, you can edit the files and run the server on you local machine!
- To play the game, visit - [https://run-sonic-run.vercel.app](https://run-sonic-run.vercel.app)
